S369 is a new 2048 bit symetric block cipher designed to resist against "quantum doomsday".
Key and block size is equal, iterated block cipher with 12 rounds. Uses key dependent substitution
boxes and permutes data on 2048 bit block (also key dependent).

S369 is slow cipher because of heavy permutation and bit shifting operations.

S369 is designed as 8 bit cipher. 

S369 generates non-linear properties from initial 2, 4 bit "GOST-borrowed" S-boxes.

This project is a test suite of S369 algorithm written in Object Pascal.
